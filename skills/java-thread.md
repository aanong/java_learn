# 单例模式
## 常见单例模式
- 饿汉式（静态常量）
- 饿汉式（静态代码块）
- 懒汉式（线程不安全）
- 懒汉式（同步方法）
- 双重检查锁定（DCL - Double-Checked Locking）
- 静态内部类
- 枚举单例（最安全）
- 使用 CAS 实现（无锁）
- 单例模式的破坏与防护
- 序列化攻击

- Spring 中的单例 Bean
- Runtime 类
- 日志框架 Logger
- 数据库连接池
- 配置管理器
- 其他业务场景应用
- 各种实现方式对比
